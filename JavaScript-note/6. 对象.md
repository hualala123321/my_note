# 对象
+ 对象是封装了一些属性和方法的集合。
+ 对象是JavaScript的一个基本数据类型，是一种复合值，它将很多值（原始值或者其他对象）聚合在一起，可通过名字访问这些值。即属性的无序集合。
+ JS中，可以将对象分为“内置对象”、“宿主对象”和“自定义对象”三种。
    - 内置对象:系统所提供的对象;如Object、Array、Math、Date等等。
    - 宿主对象：JS所运行的环境提供的对象，可以理解为：浏览器提供的对象。所有的BOM和DOM对象都是宿主对象。
    - 自定义对象：自定义构造函数所创建的对象。
